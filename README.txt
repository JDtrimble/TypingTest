 Задумка проекта проста: иногда(мне достаточно часто) интересно в целях улучшения скорости и аккуратности печати, или проверки новой клавиатуры, к примеру. Для этого я сделал вот эту небольшую програмку, которую если постараться можно даже в 1 файл запихать. (Я так не сделал для лучшей ориентации в коде)

Размеры: ~150 строчек главная часть; ~50 строчек работа с базой данных; ~150 строчек интерфейс

Особенности: 
1.CAPS-LOCK я не реализовал, по причине серьезного усложнения кода(у меня было достаточно мало времени на написание проекта) и по причине того что при быстрой печати его не используют. 
2. Все тексты были найдены на открытых ресурсах и не имеют праввообладателя.
3. Можно спокойно реализовать русский язык дополнительно без серьезных изменений в коде(и при наличии мотивации)
4. Хоть я и переводил файл main.exe как было написано в гайде из лицея, он у меня без иконки и мне кажется переведен коряво.
5. В файле records.json хранится вся информация по прохождениям тестов ранее.
6. В файле tests.db хранятся тексты для тестирования.
